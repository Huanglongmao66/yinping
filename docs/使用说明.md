## 一、快速开始

### 1.1 安装与启动

#### Windows 系统
1. 双击 `YinYuePlayer-Setup.exe` 安装程序
2. 按照安装向导完成安装
3. 双击桌面图标或从开始菜单启动

#### macOS 系统
1. 打开 `YinYuePlayer.dmg` 镜像文件
2. 将 `音悦播放器` 拖拽到「应用程序」文件夹
3. 从启动台或应用程序中启动

#### Linux 系统
1. 解压 `YinYuePlayer.tar.gz` 到指定目录
2. 进入目录后执行 `./YinYuePlayer` 启动
3. 或双击 `YinYuePlayer` 可执行文件

> **注意**：首次运行需要安装 Java 17 或更高版本。

### 1.2 添加音乐

#### 方法一：文件菜单
1. 点击顶部菜单栏「文件」→「打开文件」
2. 选择要播放的音频文件
3. 点击「打开」即可开始播放

#### 方法二：拖拽添加
1. 打开存放音乐的文件夹
2. 将音频文件或文件夹直接拖拽到播放器窗口
3. 音乐将自动添加到播放列表

#### 方法三：添加文件夹
1. 点击「文件」→「打开文件夹」
2. 选择存放音乐的文件夹
3. 软件会自动扫描文件夹内所有音频文件

### 1.3 首次使用建议

1. **添加音乐库**：将常用音乐文件夹添加到音乐库，方便管理
2. **创建播放列表**：根据不同场景创建播放列表
3. **设置主题**：选择喜欢的界面主题（亮色/暗色）
4. **配置快捷键**：根据使用习惯设置快捷键

## 三、播放控制

### 3.1 基本播放操作

#### 播放/暂停
- 点击底部控制栏中间的播放/暂停按钮
- 按空格键 `Space`
- 按 `Ctrl + P`

#### 停止
- 点击「播放」→「停止」
- 按 `Ctrl + T`

#### 上一曲/下一曲
- 点击控制栏的上一曲/下一曲按钮
- 按 `Ctrl + ←`（上一曲）/ `Ctrl + →`（下一曲）

### 3.2 进度控制

#### 拖动跳转
1. 在底部进度条上按住鼠标左键
2. 拖动到想要的位置
3. 松开鼠标即可跳转

#### 精准跳转
- 悬停在进度条上会显示时间提示
- 点击进度条任意位置跳转到该位置

### 3.3 音量控制

#### 调节音量
1. 点击底部右侧的音量滑块
2. 左右拖动调节音量大小
- 或按 `Ctrl + ↑` / `Ctrl + ↓` 微调音量

#### 静音
- 点击音量图标切换静音
- 按 `Ctrl + M`

### 3.4 播放模式

点击控制栏的播放模式按钮切换模式：

| 模式 | 图标 | 说明 |
|------|------|------|
| 顺序播放 | ▶▶ | 按列表顺序播放，播完停止 |
| 列表循环 | 🔁 | 循环播放整个列表 |
| 单曲循环 | 🔂 | 重复播放当前歌曲 |
| 随机播放 | 🔀 | 随机播放列表中的歌曲 |

> **提示**：可在「播放」菜单中选择播放模式。

## 四、播放列表管理

### 4.1 查看播放列表

- 点击右下角「播放列表」按钮
- 右侧面板显示当前播放列表
- 当前播放的歌曲会高亮显示并带有音乐图标

### 4.2 添加歌曲

#### 单首添加
1. 点击「文件」→「打开文件」
2. 选择音频文件
3. 歌曲自动添加到播放列表末尾

#### 批量添加
- 拖拽多个文件到播放列表
- 打开文件夹批量添加

### 4.3 调整播放顺序

#### 拖拽排序
1. 在播放列表中按住歌曲
2. 拖动到目标位置
3. 松开完成排序

#### 右键菜单
右键点击歌曲，选择：
- 上移
- 下移
- 移到顶部
- 移到底部

### 4.4 删除歌曲

右键点击歌曲 → 选择「从列表移除」，或按 `Delete` 键。

> **注意**：此操作仅从播放列表中移除，不会删除本地文件。

### 4.5 管理播放列表

#### 新建播放列表
1. 点击左侧导航栏底部的「+ 新建」按钮
2. 输入播放列表名称
3. 点击确定

#### 保存播放列表
1. 点击「文件」→「保存播放列表」
2. 选择保存位置和文件名
3. 播放列表文件格式为 `.m3u` 或 `.json`

#### 加载播放列表
1. 点击「文件」→「打开文件」
2. 选择播放列表文件
3. 播放列表自动加载

#### 删除播放列表
- 在左侧导航栏右键点击播放列表
- 选择「删除」
- 确认删除操作

#### 重命名播放列表
- 在左侧导航栏右键点击播放列表
- 选择「重命名」
- 输入新名称

## 五、音乐库管理

### 5.1 添加音乐库文件夹

1. 点击左侧导航栏「音乐库」
2. 点击「添加文件夹」按钮
3. 选择存放音乐的文件夹
4. 软件自动扫描并索引音乐文件

### 5.2 浏览音乐

#### 按全部歌曲浏览
- 点击「音乐库」查看所有歌曲
- 支持网格视图和列表视图切换

#### 按歌手浏览
1. 点击「歌手」
2. 查看所有歌手列表
3. 点击歌手查看该歌手的所有歌曲

#### 按专辑浏览
1. 点击「专辑」
2. 查看所有专辑封面
3. 点击专辑查看专辑内歌曲

### 5.3 搜索音乐

1. 点击顶部搜索框，或按 `Ctrl + F`
2. 输入关键词（歌名、歌手、专辑）
3. 搜索结果实时显示

### 5.4 刷新音乐库

- 点击「工具」→「刷新音乐库」，或按 `F5`
- 软件重新扫描音乐文件夹，更新音乐库

## 六、歌词显示

### 6.1 加载歌词

#### 自动加载
- 如果音频文件同目录下有同名 `.lrc` 文件，软件会自动加载
- 例如：`夜曲.mp3` 对应 `夜曲.lrc`

#### 手动加载
1. 播放歌曲时，右键点击歌词区域
2. 选择「加载歌词文件」
3. 选择对应的 `.lrc` 文件

### 6.2 歌词显示

- 歌词在中间内容区底部显示
- 当前播放的歌词行高亮显示
- 歌词随播放进度自动滚动

### 6.3 歌词设置

右键点击歌词区域：
- 字体大小调整
- 歌词颜色设置
- 显示/隐藏翻译（如果有）

### 6.4 桌面歌词（可选功能）

开启方式：
- 点击「视图」→「桌面歌词」
- 桌面歌词悬浮显示在屏幕上
- 可拖动调整位置

## 八、均衡器设置

### 8.1 打开均衡器

- 点击控制栏右侧「更多」→「均衡器」
- 或按 `Ctrl + E`
- 或点击「视图」→「显示均衡器」

### 8.2 使用预设

1. 点击「预设」下拉菜单
2. 选择合适的预设：
   - 流行
   - 摇滚
   - 古典
   - 爵士
   - 人声
   - 重低音
   - 自定义

### 8.3 手动调节

1. 勾选「启用均衡器」
2. 拖动各频段滑块调节增益
3. 频段从左到右：低音 → 高音
4. 调节范围：-12dB 到 +12dB

#### 10段均衡器频段
| 序号 | 频率 | 说明 |
|------|------|------|
| 1 | 32Hz | 超低音 |
| 2 | 64Hz | 低音 |
| 3 | 125Hz | 中低音 |
| 4 | 250Hz | 中低音 |
| 5 | 500Hz | 中音 |
| 6 | 1kHz | 中音 |
| 7 | 2kHz | 中高音 |
| 8 | 4kHz | 高音 |
| 9 | 8kHz | 高音 |
| 10 | 16kHz | 超高音 |

### 8.4 保存自定义预设

1. 调节好各频段后
2. 点击「保存」按钮
3. 输入预设名称
4. 以后可直接从预设列表中选择

### 8.5 重置均衡器

点击「重置为默认」按钮，所有频段恢复为 0dB。

- 继续：点击「继续」按钮

### 9.4 保存录音

1. 停止录音后，点击「保存」
2. 选择保存位置
3. 输入文件名
4. 点击保存

## 十二、常见问题

### 12.1 播放相关

**Q: 为什么有些音频文件无法播放？**
> A: 请确认文件格式是否受支持。目前支持 MP3、WAV、FLAC、OGG 等常见格式。如果文件损坏或编码特殊，可能无法播放。

**Q: 播放时没有声音怎么办？**
> 1. 检查系统音量是否开启
> 2. 检查播放器音量是否调到最低或静音
> 3. 检查音频输出设备是否正确
> 4. 尝试播放其他音频文件确认是否是文件问题

**Q: 播放卡顿怎么办？**
> 1. 关闭其他占用资源较多的程序
> 2. 尝试降低可视化效果复杂度
> 3. 检查音频文件是否损坏

### 12.2 歌词相关

**Q: 为什么不显示歌词？**
> 1. 确认是否有对应的 LRC 歌词文件
> 2. 歌词文件名是否与歌曲名一致
> 3. 歌词文件是否和歌曲在同一目录
> 4. 检查是否关闭了歌词显示

**Q: 歌词不同步怎么办？**
> 可以手动调整歌词偏移：右键歌词区域 → 歌词偏移 → 提前/延后。

### 12.3 音乐库相关

**Q: 添加的音乐在音乐库中找不到？**
> 1. 尝试刷新音乐库（按 F5）
> 2. 确认文件夹路径正确
> 3. 检查文件格式是否受支持

**Q: 如何删除音乐库中的歌曲？**
> 音乐库只是索引，要从音乐库移除歌曲，只需从原文件夹删除文件，然后刷新音乐库即可。

### 12.4 界面相关

**Q: 界面显示异常怎么办？**
> 尝试在设置中切换主题，或重启软件。

**Q: 如何更换主题颜色？**
> 点击「视图」→「主题」，选择「亮色」或「暗色」主题。

### 12.5 其他问题

**Q: 软件启动不了怎么办？**
> 1. 确认已安装 Java 17 或更高版本
> 2. 尝试以管理员身份运行
> 3. 查看日志文件排查错误

**Q: 配置文件和数据保存在哪里？**
> - Windows: `%APPDATA%/YinYuePlayer/`
> - macOS: `~/Library/Application Support/YinYuePlayer/`
> - Linux: `~/.config/YinYuePlayer/`

**Q: 如何重置所有设置？**
> 退出软件后，删除配置目录下的 `config.json` 文件，重新启动软件即可恢复默认设置。
