# 音悦音频播放器 - 软件布局设计文档

## 一、整体布局设计

### 1.1 设计理念

- **简洁现代**：采用扁平化设计风格，去除不必要的装饰元素
- **操作便捷**：常用功能触手可及，减少操作步骤
- **视觉舒适**：合理的色彩搭配，保护用户视力
- **响应式布局**：支持窗口大小自由调整

### 1.2 整体结构

主窗口采用经典的三栏布局，整体结构清晰，功能分区明确：

```
┌─────────────────────────────────────────────────────────────┐
│  菜单栏  (Menu Bar)                                         │
├──────────┬──────────────────────────────────┬───────────────┤
│          │                                  │               │
│  左侧    │         中间内容区                │   右侧面板    │
│  导航    │    (专辑封面 + 可视化 + 歌词)    │  (播放列表)   │
│  栏      │                                  │               │
│          │                                  │               │
├──────────┴──────────────────────────────────┴───────────────┤
│  底部播放控制栏  (Player Control Bar)                       │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 窗口尺寸

- **最小尺寸**：800 × 500 px
- **默认尺寸**：960 × 600 px
- **推荐尺寸**：1200 × 700 px
- **宽高比**：支持自由调整

## 二、各区域详细设计

### 2.1 顶部菜单栏 (Menu Bar)

#### 布局说明
位于窗口最顶部，提供全局菜单入口。

#### 菜单项结构

```
┌─────────────────────────────────────────────────────────────┐
│ 文件(F)  编辑(E)  播放(P)  视图(V)  工具(T)  帮助(H)       │
└─────────────────────────────────────────────────────────────┘
```

#### 文件菜单
| 菜单项 | 快捷键 | 功能说明 |
|--------|--------|----------|
| 打开文件 | Ctrl+O | 打开音频文件 |
| 打开文件夹 | Ctrl+Shift+O | 打开文件夹添加到播放列表 |
| 添加到播放列表 | - | 子菜单：选择播放列表 |
| 保存播放列表 | Ctrl+S | 保存当前播放列表 |
| 退出 | Ctrl+Q | 退出程序 |

#### 编辑菜单
| 菜单项 | 快捷键 | 功能说明 |
|--------|--------|----------|
| 搜索 | Ctrl+F | 搜索歌曲 |
| 偏好设置 | Ctrl+, | 打开设置窗口 |

#### 播放菜单
| 菜单项 | 快捷键 | 功能说明 |
|--------|--------|----------|
| 播放/暂停 | Ctrl+P / Space | 播放或暂停 |
| 停止 | Ctrl+T | 停止播放 |
| 上一曲 | Ctrl+Left | 播放上一首 |
| 下一曲 | Ctrl+Right | 播放下一首 |
| 音量增加 | Ctrl+Up | 音量+5% |
| 音量减少 | Ctrl+Down | 音量-5% |
| 静音 | Ctrl+M | 切换静音 |
| 播放模式 | - | 子菜单：顺序/单曲循环/列表循环/随机 |

#### 视图菜单
| 菜单项 | 快捷键 | 功能说明 |
|--------|--------|----------|
| 迷你模式 | Ctrl+1 | 切换迷你模式 |
| 完整模式 | Ctrl+2 | 切换完整模式 |
| 显示歌词 | Ctrl+L | 显示/隐藏歌词面板 |
| 显示均衡器 | Ctrl+E | 显示/隐藏均衡器 |
| 主题 | - | 子菜单：亮色/暗色/跟随系统 |

#### 工具菜单
| 菜单项 | 快捷键 | 功能说明 |
|--------|--------|----------|
| 格式转换 | - | 打开格式转换工具 |
| 录音 | Ctrl+R | 打开录音功能 |
| 刷新音乐库 | F5 | 重新扫描音乐库 |

#### 帮助菜单
| 菜单项 | 快捷键 | 功能说明 |
|--------|--------|----------|
| 使用说明 | F1 | 打开帮助文档 |
| 关于 | - | 显示版本信息 |

### 2.2 左侧导航栏 (Sidebar)

#### 布局说明
位于窗口左侧，宽度可调整（默认180px，最小120px）。

#### 导航项结构

```
┌─────────────┐
│  ⌛ 正在播放 │  ← 高亮显示
│  🎵 音乐库   │
│  💿 专辑     │
│  🎤 歌手     │
│  📋 歌单     │
│             │
│ ─────────── │
│  📁 我的歌单 │
│  🎧 常用歌曲 │
│  ⭐ 我喜欢   │
│  ...        │
│             │
│  [+] 新建   │
└─────────────┘
```

#### 导航项说明

| 图标 | 名称 | 功能说明 |
|------|------|----------|
| ⌛ | 正在播放 | 显示当前播放列表 |
| 🎵 | 音乐库 | 显示全部音乐 |
| 💿 | 专辑 | 按专辑分类浏览 |
| 🎤 | 歌手 | 按歌手分类浏览 |
| 📋 | 歌单 | 显示所有播放列表 |
| 📁 | 我的歌单 | 用户自定义歌单（可展开/收起） |

#### 交互设计
- 鼠标悬停显示背景高亮
- 当前选中项左侧显示竖条指示器
- 歌单支持右键菜单：重命名、删除、导出
- 底部「新建」按钮用于创建新播放列表

### 2.4 右侧播放列表面板 (Playlist Panel)

#### 布局说明
位于窗口右侧，宽度可调整（默认280px，最小200px）。

```
┌─────────────────────────┐
│ 播放列表           ⋮  ✕ │ ← 标题栏
├─────────────────────────┤
│                         │
│  1. 夜曲           ♪    │ ← 当前播放高亮
│     周杰伦  03:48       │
│  2. 七里香               │
│     周杰伦  04:59       │
│  3. 青花瓷               │
│     周杰伦  03:58       │
│  4. 稻香                 │
│     周杰伦  03:43       │
│  ...                    │
│                         │
└─────────────────────────┘
```

#### 列表项说明

每行显示内容：
- 序号
- 歌曲名（主标题）
- 歌手 - 时长（副标题，灰色小字）
- 当前播放指示（音乐图标或动画）

#### 操作功能
- 双击歌曲：播放该歌曲
- 拖拽：调整播放顺序
- 右键菜单：
  - 播放
  - 上移 / 下移
  - 移到顶部 / 移到底部
  - 从列表移除
  - 查看文件信息
  - 在文件夹中显示
- 右上角菜单按钮：列表管理操作
- 可通过拖动左边缘调整宽度

### 2.5 底部播放控制栏 (Player Control Bar)

#### 布局说明
位于窗口最底部，高度固定60px，始终显示。

```
┌─────────────────────────────────────────────────────────────────────────┐
│  ┌──┐                                                                │
│  │封│  歌曲名                                                         │
│  │面│  歌手 - 专辑                                                    │
│  └──┘                                                                │
│         ⏮️  ⏸️  ⏭️       🔀  🔁  🔂                                  │
│     01:23 ━━━━━●━━━━━━━━━━━━━ 04:56                                │
│                                                                     │
│                                             🔊 ●━━━━━━  ⏏  ⋮     │
└─────────────────────────────────────────────────────────────────────────┘
```

#### 详细元素说明

| 区域 | 元素 | 功能说明 |
|------|------|----------|
| 左侧 | 专辑封面 (50×50px) | 点击展开/收起详情，悬停显示放大预览 |
| 左侧 | 歌曲信息 | 歌名（粗体）、歌手-专辑（灰色小字） |
| 中间 | 上一曲按钮 | 播放上一首 |
| 中间 | 播放/暂停按钮 | 圆形大按钮，播放时显示暂停图标，暂停时显示播放图标 |
| 中间 | 下一曲按钮 | 播放下一首 |
| 中间 | 播放模式按钮组 | 随机播放、列表循环、单曲循环（三选一或组合） |
| 中间 | 进度条 | 显示播放进度，可拖动跳转 |
| 中间 | 时间显示 | 当前时间 / 总时长 |
| 右侧 | 音量图标 | 点击切换静音 |
| 右侧 | 音量滑块 | 调节音量大小 |
| 右侧 | 播放列表按钮 | 显示/隐藏右侧播放列表面板 |
| 右侧 | 更多按钮 | 菜单：均衡器、格式转换、录音、设置 |

#### 进度条设计
- 已播放部分：主题色填充
- 未播放部分：浅灰色
- 缓冲进度：比已播放稍浅的颜色（流媒体时显示）
- 悬停显示时间气泡提示
- 拖动时改变鼠标样式

## 五、交互动效设计

### 5.1 过渡动画

- 页面切换：300ms 淡入淡出
- 悬停效果：200ms 颜色渐变
- 按钮点击：缩放 0.95 再恢复
- 弹窗出现：从上方滑入 + 淡入

### 5.2 播放状态动画

- 专辑封面：播放时缓慢旋转（60秒一圈），暂停时停止
- 播放按钮：播放/暂停状态切换平滑过渡
- 频谱可视化：60fps 流畅动画

### 5.3 反馈效果

- 按钮点击有视觉反馈（颜色变化 + 轻微缩放）
- 滑块拖动实时更新
- 操作成功/失败有Toast提示
