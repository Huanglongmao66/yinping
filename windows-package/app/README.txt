音悦播放器 YinYue Player v1.0.0
================================

系统要求
--------
- Windows 10/11 (64位)
- 最少 4GB 内存
- 200MB 可用磁盘空间

目录结构
--------
YinYuePlayer/
  start.bat          - 启动程序
  start-debug.bat    - 调试模式启动
  runtime/           - Java 运行时 (JDK 17 + JavaFX)
  app/
    yinyue-player.jar    - 主程序
    lib/
      gson-2.10.1.jar    - Gson 依赖

启动方式
--------
双击 start.bat 即可启动音悦播放器。

如遇到问题，可使用 start-debug.bat 查看详细错误信息。

功能简介
--------
- 支持 MP3/WAV/FLAC/OGG/AAC/M4A 格式
- 播放列表管理与智能推荐
- 10频段均衡器与多种音效预设
- 桌面歌词悬浮窗
- 迷你播放器模式
- 网络电台
- 音频录制
- 5套主题切换
- 播客/RSS订阅
- HTTP远程控制
- 云同步备份

快捷键
------
Space      播放/暂停
Ctrl+←     上一曲
Ctrl+→     下一曲
Ctrl+E     均衡器
Ctrl+L     桌面歌词
M          静音
F11        全屏
